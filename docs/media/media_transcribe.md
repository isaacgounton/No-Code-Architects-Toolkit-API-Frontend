# Media Transcription Endpoint

## 1. Overview

The `/v1/media/transcribe` endpoint is part of the Flask API application and is responsible for transcribing media files (audio or video) into text format. It supports various options such as generating SRT subtitles, including word-level timestamps, and translating the transcription into different languages. The endpoint can return the transcription results directly or upload them to cloud storage and provide the URLs.

## 2. Endpoint

- **URL Path**: `/v1/media/transcribe`
- **HTTP Method**: `POST`

## 3. Request

- `x-api-key` (required): The API key for authentication.
